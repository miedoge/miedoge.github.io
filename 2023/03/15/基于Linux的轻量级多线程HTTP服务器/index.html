基于Linux的轻量级多线程HTTP服务器

  1. 简介：此项目是基于Linux的轻量级多线程Web服务器，应用层实现了一个简单的HTTP服务器，支持静态资源访问与动态消息回显。

  2. 由来：这个项目是基于C++实现的HTTP1.0服务器，虽然是烂大街的webserver，不过是自己动手做的，感觉还是有意义的就放上来了。

  3. 编译：

    • 本人的Linux环境如下

    • 开发环境是vscode加这些插件

    • 第一步，先克隆本项目,然后给予足够的权限

    • cd  httpdocs

sudo chmod 600 index.html

sudo chmod 600 post.html

sudo chmod +X post.cgi

    • 编译执行：
      g++ http.cpp -D_REENTRANT~ -o http1 -lpthread

    • ./http1 8888

    • 然后访问 127.0.0.1:8888/index.html 即可.
